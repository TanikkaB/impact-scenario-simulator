# 🔮 Impact Scenario Simulator

A tool for modelling how operational changes may affect outcomes and performance.

This project is part of the **Impact Analytics Suite**, a portfolio of decision-support tools focused on impact, programme, and data strategy workflows.

---

## 🎯 Overview

Decision-making often involves understanding potential outcomes of changes. This tool allows users to simulate different scenarios.

---

## ✨ Key Features

- Adjustable scenario inputs  
- Baseline vs projected comparison  
- Outcome estimation  
- Cost impact modelling  
- Scenario summary insights  

---

## 🧠 Why I Built It

I built this to demonstrate how simple scenario modelling can support structured decision-making.

---

## 👥 Example Use Case

- Testing impact of increased participation  
- Exploring changes in completion rates  
- Supporting strategic planning  

---

## 🗂 Expected Data Structure

- programme  
- participants  
- completion_rate  
- satisfaction_score  
- cost_per_participant  

---

## 🧪 Demo Mode

Includes synthetic dataset designed to show meaningful scenario changes.
